### [Google API Keys Weren't Secrets. But then Gemini Changed the Rules. ◆ Truffle Security Co.](https://trufflesecurity.com/blog/google-api-keys-werent-secrets-but-then-gemini-changed-the-rules)

> Google spent over a decade telling developers that Google API keys (like those used in Maps, Firebase, etc.) are not secrets. But that's no longer true.

#### AI generated summary

Truffle Security identified a vulnerability where Google API keys, initially public identifiers, gained unintended access to sensitive Gemini API endpoints. After reporting the issue, Google took steps to address it, including reclassifying it as a bug and implementing measures to prevent similar issues. The article advises Google Cloud users on how to check for exposure and highlights the broader security implications.
